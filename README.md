## Repository Structure

```
MicroPress-plugins/
├── registry.json          # Plugin registry
├── sdk/                   # Plugin SDK
│   ├── src/
│   ├── dist/
│   └── README.md
└── examples/              # Example plugins (coming soon)
```

## Plugin Registry

The `registry.json` file contains metadata for all available plugins. Plugins are distributed via GitHub releases.
